Momii Dot Explorer

Chọn folder / thả file / paste text → tự chunk thành dots + liên kết

Inputs

Cách dùng nhanh
1) Bấm Load folder và chọn đúng thư mục MOMII/ (hoặc bấm Add files / thả file vào đây).
2) Tool sẽ chia nhỏ (chunk) nội dung thành nhiều dots; dots liên kết qua keywords (entities).
3) Click 1 dot để xem chi tiết + các dots liên quan.
Refresh chỉ rebuild từ những file bạn đã chọn (nếu file trên disk thay đổi, bạn cần chọn lại).
Lưu ý: file này dùng CDN (Cytoscape, pdf.js, PapaParse) nên cần internet để tải thư viện; chọn folder (webkitdirectory) hoạt động tốt nhất trên Chrome/Edge.
