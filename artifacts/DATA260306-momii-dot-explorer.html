Momii Dot Explorer

Object-first graph → click object để xem evidence

Inputs

Cách dùng
1) Bấm Load folder MOMII và chọn thư mục MOMII/.
2) Mặc định graph chỉ hiện Objects (Templates/Personas/Collab/Funnel/Competitors/Auto).
3) Click 1 object → vào Focus view (chỉ hiện evidence liên quan).
4) Run OCR (tuỳ chọn) để đọc text trong ảnh và link objects bằng OCR. (Có thể cần chạy qua localhost).
Lưu ý: file này dùng CDN (Cytoscape, pdf.js, PapaParse) nên cần internet để tải thư viện.
